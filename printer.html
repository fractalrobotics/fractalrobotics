Fractal 5 Pro

The Fractal 5 Pro is an open source benchtop multidirectional 5-axis 3D printer.

Design Features

  • Compatible with Slicer
  • CoreXY Motion System
  • Direct Drive
  • Heated Build Plate
  • Fully Enclosed
  • 300mm Diameter X 250mm Build Height
  • Auto Bed Leveling
  • Rigid 30x30mm Aluminum Extrusions
  • Backwards Compatible with 3-Axis FDM

For more information and access to the design, visit the Fractal 5 Pro Github repository.
